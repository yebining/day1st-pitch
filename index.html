인플루언서 발굴부터 컨택·미팅까지, Day1st.

핵심 기능 · 생산성 증대

구현한 것 4가지

01

키워드 티어별 인플루언서

키워드 입력하면 1 / 1.5 / 2 / 번외 티어로 자동 정리. 각 영상 클릭 시 유튜브 원본 바로 확인.

티어 기준은 직접 설정 가능 — 본인이 원하는 구독자/조회수 임계값 직접 입력하거나, 본보기 강사 YouTuber를 AI에게 분석시켜 추천 임계값을 자동 생성.

02

바로 컨택 메일 전송

영상 괜찮으면 바로 컨택 페이지 → AI 개인화 메일 생성 → 편집 → Gmail로 실발송. 개인 Gmail 서명(명함)도 자동 첨부.

03

미팅 캘린더 인비 자동 전송

UI 안에서 날짜·시간 입력 → Google Calendar 이벤트 자동 생성 + 참석자 초대 + Meet 링크.

04

컨택 연사 DB · 전사 공유

Day1st으로 컨택한 연사는 전사 공유 DB에 자동 기록. 응답 대기 → 회신 완료 → 미팅 확정으로 상태 자동 전이. 중복 컨택을 UI 단계에서 사전 차단.

그래서 뭐가 좋아졌나

정리하면 5줄

  • 1 유튜브 맨날 뒤지던 걸 한 화면에
  • 2 메일·미팅 잡는 걸 다른 페이지 안 넘어가고
  • 3 답장 오면 같은 화면에서 바로 답신·미팅 확정
  • 4 영상 댓글 반응까지 한 화면에서 분석
  • 5 전사 중복 컨택 사전 차단

STEP 01 / 08

로그인

사용을 원하시면 김예빈에게 슬랙 DM 주세요!

권한 부여해드리겠습니다.

(현재 권한 부여 없이도 로그인 가능하도록 디벨롭 중)

절차

  1. 1홈 화면 우측 상단 “Google로 로그인” 버튼 클릭
  2. 2본인 회사 계정 (yebin.kim@day1company.co.kr 같은 형식)으로 로그인
  3. 3Gmail / Calendar 권한 동의 (메일 발송 · 캘린더 인비 자동 보내기 위함)
  4. 4완료 — 헤더에 본인 프로필 사진 + 이름 표시

알아두면 좋은 점

  • · 로그인 1회 영구 유지 — 토큰 자동 갱신 (1시간마다)
  • · 권한 회수 또는 비밀번호 변경 시에만 헤더에 “🔄 재로그인 필요” 빨간 버튼이 표시됩니다 → 한 번 클릭으로 복구
  • · Gmail 메일 / Calendar 일정은 본인 계정으로만 발송 — 본인 명의로 나갑니다
STEP 02 / 08

티어 기준 설정

홈 → 우측 상단 ⚙️ 티어 기준 설정

방법 A

직접 입력

  1. · 1티어 / 1.5티어 / 2티어 각각 구독자 수 / 평균 조회수 최소 입력
  2. · 카테고리에 맞게 자유롭게 (테크 / 플라워 / 디자인 / 일러스트 등)
  3. · 저장 누르면 검색 · 알람 결과에 즉시 반영

기본값(테크 기준)이 깔려 있어서, 안 바꾸면 자동으로 그게 적용

방법 B · AI

🎯 좋은 강사 분석

  1. · “이 분 같은 강사 더 찾고 싶다” 본보기 YouTuber 1~8명 URL 입력
  2. · AI로 분석하기 → 10~20초 후 결과:
    • 📌 공통점 / 특징
    • 🎬 어떤 콘텐츠를 올리는가 (주제·포맷·톤)
    • 💡 강사로서의 강점
    • 🎚 추천 티어 임계값 (구독자 / 조회수)
  3. · 이대로 폼에 채우기 클릭 → 자동 입력 → 저장
⚠️ AI 분석은 Gemini 무료 한도 (분당 15회 · 하루 1500회)가 초과되면 일시 실패할 수 있습니다. 잠시 기다렸다가 재시도하거나, 한국시간 자정 이후 다시 가능.
STEP 03 / 08

키워드 검색

티어별 인플루언서 + 대표 영상 + 댓글 반응까지 한 화면에

01

홈 검색창에 키워드 입력 → 검색

예: “AI 생산성”, “n8n”, “프롬프트 엔지니어링”

02

결과가 1티어 / 1.5티어 / 2티어 / 번외로 자동 분류

STEP 2에서 정한 기준 그대로 적용됩니다. 카드 우측에 구독자수 · 평균 조회수 · 알림 배지(🔥 떠오름 등)

03

관심 가는 인플루언서 카드 클릭 → 컨택 페이지

좌측에 프로필 + 키워드 매칭된 영상 목록 표시. 영상 클릭 시 YouTube 원본 새 탭으로 열림

04 · 핵심

💬 대표 영상 댓글 반응 자동 분석

매칭된 영상의 인기 댓글 TOP 10을 좋아요 순으로 보여주고, 가장 공감된 댓글 1건은 강조 표시. 평균 좋아요 · 총 대댓글 통계까지.
→ “이 영상에 사람들이 어떻게 반응했는지”를 직접 영상 안 봐도 즉시 파악.

STEP 04 / 08

컨택 메일 발송

컨택 페이지 우측에서, 내 Gmail로 진짜 발송

1

제안 강의 주제 입력

예: “AI 에이전트 실전 개발”

2

받는 사람 이메일 입력

강사 본인 메일 주소

3

메일 제목 (자동 채워짐, 수정 가능)

기본: [패스트캠퍼스] OOO님께 강의 제안 드립니다

4

“AI 메일 생성” → 본문 자동 작성

인플루언서 콘텐츠 · 키워드를 반영한 개인화 메일. 미리보기에서 그대로 또는 수정 후 발송 가능

5

“Gmail로 발송하기” → 실제 발송

로그인한 본인 Gmail 발신함에 그대로 기록됩니다. Gmail에 등록된 서명(명함)도 자동으로 메일 끝에 첨부되어 발송 — 별도 설정 불필요

STEP 05 / 08

회신 · 미팅 · 답장

컨택 페이지 한 화면에서 전부 끝남

🔔

상대 답장 자동 감지

발송한 스레드를 10초마다 폴링. 답장이 오면 컨택 페이지 채팅창에 자동으로 떠. “Gmail 답변 수신 감지” 시스템 메시지 + 상대 답변 내용 표시.

📅

미팅 인비 한 번에 발송

  1. · 채팅창 위 날짜 · 시간 입력
  2. · 📅 확정 & 초대 발송 클릭
  3. · Google Calendar 이벤트 자동 생성 + 참석자 초대장 + Google Meet 링크
  4. · 컨택 상태가 “미팅 확정” 으로 자동 전환
✉️

추가 답장도 같은 채팅창에서

채팅 입력창에 메시지 입력 → 전송 → 같은 Gmail 스레드로 실제 답신 발송. 새 메일창을 따로 열 필요가 없습니다.
Gmail 서명도 답신마다 자동 첨부됩니다.

STEP 06 / 08

🙋 내 컨택

홈 우측 상단 → 🙋 내 컨택

한눈에 보이는 것

  • · 본인이 발송한 모든 컨택 리스트 (전사가 아닌 본인 것만)
  • · 상태 카운트 카드: 응답 대기 · 회신 완료 · 미팅 확정 각 N건
  • · 행마다: 강사명 · 플랫폼 · 주제 · 발송일 · 현재 상태 뱃지

대화 이어가기 ↗

행 클릭하면 해당 컨택 페이지로 복귀. Gmail 스레드 자동 동기화로 이전에 주고받은 메시지 전부 채팅창에 펼쳐짐. 거기서:

  • · 새 답장 발송 (같은 스레드에)
  • · 미팅 인비 추가 발송
  • · 진행 상황 그대로 이어서 작업
💡 메일 앱·시트·CRM 안 옮겨다녀도 됩니다. 내 컨택 페이지가 곧 진행 중인 컨택 워크보드.
STEP 07 / 08

📋 전사 컨택 기록

홈 우측 상단 → 📋 전사 컨택 기록 · 중복 컨택 사전 차단

01

전사 공유 DB

Day1st을 사용한 모든 동료들의 컨택 기록 통합 (Supabase). 누가 언제 누구에게 어떤 주제로 보냈는지.

02

상태 자동 전이

  • · 발송 직후: 응답 대기 (회색)
  • · 답신 감지: 회신 완료 (오렌지)
  • · 인비 발송: 미팅 확정 (초록)

한 번 미팅 확정되면 더 이상 역행 안 함

03

중복 컨택 사전 차단

검색 결과의 인플루언서 카드에 이미 컨택된 사람은 표시. 누가 어떤 상태로 진행 중인지 보고 중복 발송 전에 동료와 상의.

🔍 검색창에서 강사명 / 발송자 / 주제로 필터링 가능. 10초마다 자동 새로고침되고, 누군가 새로 발송 / 회신 받으면 즉시 반영.

STEP 08 / 08

💬 커뮤니티

홈 우측 상단 → 💬 커뮤니티 · 사용자끼리 자유 채팅

무엇을 할 수 있나

  • · 강사 정보·인사이트 공유 — “이 분 컨택해도 될까요?”
  • · 카테고리별 노하우 — 디자인·AI·플라워 등 분야별 팁
  • · 실시간 도움 요청 — 메일 카피·미팅 톤 의견 묻기
  • · 잡담도 OK

동작 방식

  • · 입력창에 메시지 → Enter 또는 전송 → 즉시 전사 공유
  • · 4초마다 자동 새로고침 — 다른 동료 메시지가 자동으로 흘러 옴
  • · 내 메시지는 레드, 다른 동료는 회색
  • · 프로필 사진 + 이름 + 시간 함께 표시
  • · 로그인된 day1 멤버만 참여
💡 다른 채팅 도구로 옮겨다닐 필요 없이, Day1st 안에서 컨택 작업하면서 동시에 동료와 대화. 강사명 / 주제 / URL 그대로 붙여 넣고 의견 받기.
