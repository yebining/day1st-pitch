Day1st

1st. 가장 먼저 찾고,
Day1. 하루 만에 섭외합니다

인플루언서 발굴부터 컨택·미팅까지, Day1st.

핵심 기능 · 생산성 증대

구현한 것 4가지

01

키워드 티어별 인플루언서

키워드 입력하면 1 / 1.5 / 2 / 2.5 티어로 자동 정리. 각 영상 클릭 시 유튜브 원본 바로 확인.

02

바로 컨택 메일 전송

영상 괜찮으면 바로 컨택 페이지 → AI 개인화 메일 생성 → 편집 → Gmail로 실발송.

03

답장 오면 Slack 즉시 알림

INBOX 10초 폴링. 답변 수신되면 Slack DM으로 자동 알림. 메일 수시로 여는 습관을 없앤다.

04

미팅 캘린더 인비 자동 전송

UI 안에서 날짜·시간 입력 → Google Calendar 이벤트 자동 생성 + 참석자 초대 + Meet 링크.

추가 기능

그 외에도

⚡

트렌드 연사 자동 알림

저장된 키워드로 조회수 10,000+ 영상이 올라오면 Slack 자동 감지. 신규 인플루언서를 경쟁사보다 먼저.

📋

컨택 연사 DB

Day1st으로 컨택한 연사는 전사 기록으로 남는다. 중복 컨택을 UI 단계에서 사전 차단.

그래서 뭐가 좋아졌나

정리하면 5줄

  • 1 유튜브 맨날 뒤지던 걸 한 화면에
  • 2 메일·미팅 잡는 걸 다른 페이지 안 넘어가고
  • 3 답장 오면 늦지 않게 바로 알림
  • 4 새 인플루언서 등장 시 즉시 감지
  • 5 전사 중복 컨택 사전 차단
